Speed up your WordPress website with Site-Fastify. Caching, asset optimization, lazy loading, and database cleanup made simple and effective.  

== Description ==  
**Site-Fastify** is your go-to solution for improving the performance and speed of your WordPress website. Designed to offer powerful optimization features while remaining easy to use, this plugin ensures that your website runs at top speed without any unnecessary complications.  

### Features  

**Caching**  
- **Page Caching:** Basic configuration for caching pages for faster loading.  
- **Browser Caching:** Static asset caching for improved load times.  

**Asset Optimization**  
- **CSS/JS Minification:** Reduces file sizes by removing unnecessary spaces, comments, and formatting.  
- **HTML Minification:** Simplifies HTML files for quicker rendering.  
- **Lazy Loading for Images:** Loads images as users scroll down the page, saving bandwidth and improving performance.  

**Database Optimization**  
- **Post Revisions Cleanup:** Deletes old post revisions to reduce database size.  
- **Trash and Spam Cleanup:** Removes unnecessary data like trashed posts and spam comments.  

**Performance Analysis**  
- **Page Speed Score:** Provides basic performance metrics to evaluate your site’s loading speed.  
- **Optimization Suggestions:** Actionable recommendations to further improve site speed and performance.  

### Why Choose Site-Fastify?  
- Lightweight and beginner-friendly.  
- Focuses on essential optimization techniques.  
- No coding required—simple setup and configuration.  

Give your WordPress site the performance boost it deserves with Site-Fastify!  

== Installation ==  

1. Upload the `Site-Fastify` folder to the `/wp-content/plugins/` directory or install the plugin directly through the WordPress plugin repository.  
2. Activate the plugin through the ‘Plugins’ menu in WordPress.  
3. Go to the Site-Fastify settings page to configure the plugin according to your needs.  

== Frequently Asked Questions ==  

= Does Site-Fastify require any technical knowledge to use? =  
Not at all! Site-Fastify is designed to be user-friendly and requires no coding knowledge.  

= Is Site-Fastify compatible with other caching plugins? =  
While Site-Fastify can work alongside some optimization tools, it’s recommended to use only one caching plugin at a time to avoid conflicts.  

= Will Site-Fastify slow down my admin dashboard? =  
No, Site-Fastify is optimized to run efficiently in the background without affecting your WordPress admin performance.  

= Can I use Site-Fastify with any theme or plugin? =  
Yes, Site-Fastify is compatible with most themes and plugins.
